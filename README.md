# Simple Minecraft Launcher

> [!WARNING]
> This is a work in progress! Nothing works at the moment.

Launches Minecraft.

## Build

1. Clone the project

```
git clone https://github.com/anthocodes99/simple-mc-launcher.git
```

2. Install packages

```
pnpm install
```

3. Build for your distro

_Windows_
```
pnpm build:win
```

_Linux_
```
pnpm build:linux
```

_macOS_
```
pnpm build:mac
```

The application output is in `/dist`
